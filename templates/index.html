How it works

Record a short clip, save it to the server, and run analysis which slices the audio into fixed-length chunks, converts to spectrograms, and runs a classifier per chunk. The UI aggregates results into an easy-to-read summary.

Quick Start
  1. Open Start Diagnosis.
  2. Choose your input and press Start to preview the waveform.
  3. Record, then Save to upload and optionally analyze.
  4. Visit Saved Recordings to view results.
Supported Formats

Browser capture uses WebM/Opus. Manual uploads accept WAV/MP3/WebM. For best results use WAV (PCM).

Recording tips

  • Place your stethoscope or mic firmly for consistent signals.
  • Record at least 3–5 seconds per site to capture cycles.
  • Reduce background noise and avoid touching the microphone.
